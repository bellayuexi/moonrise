# 月升月落 · 全年月相

一个纯静态网页：查询任意地点、任意年份每一天的**月升 / 月落时间**、**月相**，以及**面向正北时月亮所在的方位与高度**，并为当天月相配一首中国古典诗词。

天文数据全部由浏览器**本地实时计算**（基于 [SunCalc](https://github.com/mourner/suncalc) 的天文算法），页面里不存任何预生成的数据表，**无需后端、无需联网**，可离线使用。任意年份都能计算，不限于某一年。

## 功能

- **天空穹顶**：中央半圆代表天空，弧上是 24 小时刻度，标出月亮当天走过的轨迹与升 / 落位置
- **观测罗盘**：输入任意时刻，实时显示月亮的方位角、地平高度，以及“面向正北时它在你的哪一侧（右手边 / 右前方…）”
- **全年月历**：左右两列（1–6 月 / 7–12 月）竖向排列，每天一个月相图，满月发光，点击任意一天即可切换
- **当日配诗**：按当天月相匹配一首古典诗词
- 深色星空主题，响应式布局，支持“添加到主屏幕”（PWA，可离线使用）

## 本地运行

任意静态服务器即可，例如：

```bash
python -m http.server 8777
# 浏览器打开 http://localhost:8777
```

## 目录结构

```
index.html              页面结构
css/style.css           深色主题与布局
js/app.js               核心逻辑：天空穹顶、罗盘、月相绘制、全年月历
js/poems.js             按月相分组的古诗库
vendor/suncalc.js       天文计算库（月升月落 / 月相 / 方位）
manifest.webmanifest    PWA 配置
sw.js                   Service Worker（离线缓存）
icon.svg                应用图标
```

## 精度说明

采用简化天文模型，月升月落时间与月相的精度通常在数分钟以内，适合日常观测参考；不含农历，也非天文台级高精度星历。
