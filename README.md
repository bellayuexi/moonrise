# 月升月落 · 全年月相

一个纯静态网页：查询任意地点、任意年份每一天的**月升 / 月落时间**、**月相**、以及**面向正北时月亮所在的方位与高度**，并为当天月相配一首中国古典诗词。

天文计算全部在浏览器本地完成（基于 [SunCalc](https://github.com/mourner/suncalc)），**无需后端、无需联网**，可离线使用。

## 功能

- 中央半圆天空穹顶：24 小时刻度，标出月亮当天走过的弧线与升/落位置
- 观测罗盘：输入任意时刻，实时显示月亮方位角、地平高度，以及"面向正北时它在你的哪一侧"
- 全年月历：左右两列（1–6 月 / 7–12 月），每天一个月相图，满月发光，点击切换
- 深色星空主题，响应式布局，支持"添加到主屏幕"（PWA，可离线）

## 本地运行

任意静态服务器即可，例如：

```bash
python -m http.server 8777
# 浏览器打开 http://localhost:8777
```

## 部署

推送到 `main` 分支后，GitHub Actions（`.github/workflows/deploy.yml`）会自动发布到 GitHub Pages。

在线地址：见仓库 **Settings → Pages** 或 Actions 部署输出。
